# 城コレクションデータ

## 出典

- 公益財団法人日本城郭協会「日本100名城一覧」
  - https://jokaku.jp/business/great-castles/
- 公益財団法人日本城郭協会「続日本100名城一覧」
  - https://jokaku.jp/business/great-castles-sequel/

確認日: 2026-07-18

## 利用方針

このアプリには、上記一覧から確認した番号、名称、所在地、参照URLなどの事実情報のみを収録しています。
公式ロゴ、画像、スタンプ画像、認定証画像、解説文、長文紹介文は収録していません。

公式サイト上のコンテンツはオープンデータとしての明示ライセンスが確認できないため、再利用範囲を最小限にしています。
今後、より明確なライセンスを持つ座標データや自治体公開データを利用する場合は、このファイルに出典、ライセンス、確認日、加工内容を追記してください。

## 管理対象ファイル

- `src/domain/castles/castleMaster.json`
- `scripts/update-castle-master.mjs`

固定マスターには訪問状態を書き込まず、ユーザーの訪問情報はIndexedDBの `castleVisitSummaries` と `castleVisitEvents` に保存します。
公式一覧の更新確認は `npm run update:castle-master` で行えます。既存の座標や読み仮名など、公式一覧に含まれない補足項目は可能な限り保持します。

## 座標について

現時点では、200城すべてについて同一品質で検証済みの座標出典を組み込んでいません。
そのため `latitude` / `longitude` は `null` とし、`verificationStatus` を `official_list_verified_coordinates_unverified` にしています。

座標を追加する場合は、代表点の定義、出典、ライセンス、確認日を明記し、城跡の範囲ではなくタップ・地図表示用の代表点であることを区別してください。

## 公式認定との区別

アプリ内の制覇率、スタンプ、御城印、訪問状態は個人の記録です。
公益財団法人日本城郭協会その他の団体による公式認定、公式証明、公式スタンプ帳の代替ではありません。
